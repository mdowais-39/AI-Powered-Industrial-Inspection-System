# Aerothon Industrial AI Inspection - Frontend

This is the polished, product-grade frontend for the Aerothon Industrial AI Inspection system.

## Setup

1.  Current location: `Frontend-new`
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start development server:
    ```bash
    npm run dev
    ```

## Video Assets

The Landing Page Video Showcase expects the following video files. Please place them in the `public/videos/` directory:

*   `public/videos/anomaly_detection.mp4`
*   `public/videos/size_measurement.mp4`
*   `public/videos/assembly_verification.mp4`

If these files are missing, the UI will display a styled "Visual Simulation" placeholder for each task.
